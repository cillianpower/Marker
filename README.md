# Marker

A minimalist Markdown writing app for macOS, built with SwiftUI.

Marker pairs a clean Markdown editor with front-matter support, tags, and a file explorer. It is built on top of [SwiftMarkdownEngine](https://github.com/nodes-app/swift-markdown-engine), a Swift Markdown rendering engine, and aims to be a full-on notes app replacement.

## Requirements

- macOS 14.5+
- Xcode 16+
